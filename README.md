# Prex Communis

A web-based Common Prayer application for the Daily Office, based on the Book of Common Prayer.

## Features

- **Four Daily Offices**: Morning Prayer, Midday Prayer, Evening Prayer, and Compline
- **Automatic Time-Based Selection**: The app automatically selects the appropriate office based on the time of day
- **Beautiful Typography**: Traditional liturgical design with EB Garamond and clean, readable layouts
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Easy-to-Read Color Scheme**: Soft, page-like colors that are gentle on the eyes during prayer

## Time Schedule

The app automatically selects the appropriate office based on the current time:

- **Morning Prayer**: 5 AM - 11 AM
- **Midday Prayer**: 11 AM - 2 PM
- **Evening Prayer**: 2 PM - 8 PM
- **Compline**: 8 PM - 5 AM

## Usage

Simply open `index.html` in your web browser, or visit the live site if hosted on GitHub Pages.

You can manually switch between offices at any time by clicking the navigation buttons.

## Liturgical Content

The app includes traditional texts from the Book of Common Prayer:

- Opening sentences and confessions
- The Venite (Psalm 95)
- Selected Psalms
- Canticles (Te Deum, Magnificat, Nunc Dimittis)
- The Apostles' Creed
- The Lord's Prayer
- Suffrages and Collects
- Closing blessings

## Installation

### Local Use

1. Clone this repository:
   ```bash
   git clone https://github.com/euxaristia/PrexCommunis.git
   ```

2. Open `index.html` in your web browser

### GitHub Pages

To deploy on GitHub Pages:

1. Go to your repository settings
2. Navigate to "Pages" in the left sidebar
3. Under "Source", select the `main` branch
4. Save, and your app will be live at `https://euxaristia.github.io/PrexCommunis/`

## Technology Stack
